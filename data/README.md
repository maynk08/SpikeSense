# Data

## Source

All data comes from the **Numenta Anomaly Benchmark (NAB)** — an open benchmark for real-world time-series anomaly detection.

- Repository: https://github.com/numenta/NAB
- License: AGPL-3.0
- Citation: *Lavin, A., & Ahmad, S. (2015). Evaluating Real-time Anomaly Detection Algorithms – the Numenta Anomaly Benchmark. ICMLA.*

## Files Used

All files are from the `realAWSCloudwatch` category, which contains genuine AWS CloudWatch metric streams — directly relevant to cloud infrastructure monitoring.

## Ground Truth

`combined_labels.json` contains the anomaly timestamps for all NAB files. The loader merges these labels into the DataFrames, creating a binary `label` column (0 = normal, 1 = anomaly).

## Format

Each CSV has two columns:

```
timestamp,value
2014-04-10 00:04:00,91.958
2014-04-10 00:09:00,94.798
...
```

Timestamps are at 5-minute intervals. Values are raw metric readings (no normalization applied at this stage).
